# Squad B: 학습 경험 / DAU — 송하림

> **미션**: "수업을 듣는 사람이 계속 듣게 만든다"
> **핵심 KPI**: 예약 변경률 49% → 30% / AI 기능 사용률 15% → 30% / 주간 수업 횟수

---

## 데이터 배경

- 예약 변경 **49%** (1,652명), 과다요청 **12.5%** (419명, 1인당 43회)
- AI 기능 **15%** 사용, 레슨 리뷰 **6%** 도달
- 피크타임 **21~22시** (12만 이벤트) + 12시간 예약 규칙 충돌
- 해지사유 "모든 레슨 완료" **5.2%**, "실력이 늘지 않는 것 같아요" **3.9%**

---

## 과제 상세

### 1. 예약 변경 UX 개선

> 난이도: 중 | 시기: 3M

**현상**: 예약 변경이 전체 예약의 49%. 과다요청 유저 419명이 1인당 평균 43회 변경. 이들은 오히려 열성 유저인데, 시스템이 과다요청 경고로 막는 구조.

**해결**:
- 예약 변경 플로우 간소화 (현재 여러 단계 → 원탭 변경)
- 과다요청 경고 대신 빠른 변경 UI 제공
- 변경 패턴 분석 → "자주 변경하는 시간대" 자동 추천
- 주간 시간표 방식의 예약 UI 검토

**근거 데이터**:
- 변경 1,652명(49%), 과다요청 419명
- 열성 유저를 시스템이 막는 구조는 이탈을 유발

---

### 2. 12시간 예약 규칙 완화/UX 개선

> 난이도: 중 | 시기: 3M~4M

**현상**: 수업 12시간 전까지 예약해야 하는 규칙. 피크타임(21~22시) 유저가 다음날 오전 수업을 예약하는 것이 구조적으로 불가능.

**해결**:
- 규칙 완화 (12시간 → 6시간 or 유연한 시간대별 차등)
- "내일 아침 수업" 퀵 예약 기능 도입
- 피크타임에 맞춘 예약 리마인더 ("내일 수업 아직 안 잡으셨네요")

**근거 데이터**:
- 피크타임 21~22시 (12만 이벤트)
- 해지사유 "12시간 전 예약 힘들어요" 6.3%
- 구조적으로 가장 활발한 시간에 예약이 막히는 역설

---

### 3. AI 리포트/리뷰 도달 경로 개선

> 난이도: 중 | 시기: 4M

**현상**: AI 리포트 기능이 있지만 15%만 사용. 레슨 리뷰까지 도달하는 유저는 6%에 불과. 기능은 만들어놨지만 유저가 찾아가지 못하는 구조.

**해결**:
- 수업 완료→AI 리포트 생성 자동 안내 (인앱 토스트)
- 레슨 리뷰까지 가는 퍼널 재설계
- 홈에서 "미확인 리포트" 배지/카드 노출
- 리포트 확인 시 포인트 리워드

**근거 데이터**:
- AI 리포트 15% 사용
- 레슨 리뷰 6% 도달
- 기능은 있으나 도달 경로가 없음

---

### 4. 예습 참여율 향상

> 난이도: 중 | 시기: 4M~5M

**현상**: 예습 버튼 클릭 2,159명(활성 유저의 37%)이지만, 실제 예습 완료 데이터가 부족. 예습이 수업 품질에 직결되는데 참여 동기가 약함.

**해결**:
- 예습 완료 시 포인트/리워드
- 예습→수업 연결 강화 ("예습한 내용 기반 수업")
- 예습 리마인더 앱푸시 (수업 3시간 전)
- 예습 완료율 트래킹 → 학습 진도에 반영

**근거 데이터**:
- btn-prestudy 2,159명(37%) 클릭
- 예습→수업 품질 연결 데이터 수집 필요

---

### 5. 학습 진도 시각화

> 난이도: 중 | 시기: 5M

**현상**: 유저가 자신의 실력 향상을 체감하지 못함. "실력이 늘지 않는 것 같아요"가 해지사유 3.9%, 동기저하가 14%.

**해결**:
- 주간/월간 학습 리포트 (수업 횟수, 학습 시간, 표현 습득 수)
- 연속 수업 스트릭 (게이미피케이션)
- 레벨 진행도 대시보드
- 과거 수업 대비 향상 포인트 하이라이트

**근거 데이터**:
- 해지사유 "실력이 늘지 않는 것 같아요" 3.9%
- 동기저하 14%
- 성장 체감이 없으면 3개월차에 이탈

---

### 6. 콘텐츠 확장 + AI Mini Lessons

> 난이도: 높 | 시기: 5M~6M

**현상**: 장기 유저가 콘텐츠를 소진함. "모든 레슨 완료, 수업할 자료 없음"이 해지사유 5.2%. 정규 수업 외 학습 경로가 부족.

**해결**:
- 중급 토픽 확장 (EN 중급 콘텐츠 확대)
- AI Mini Lessons 도입 (짧은 형태의 AI 기반 학습)
- 정규 수업 외 학습 경로 제공 (하루 5분 미니 학습)
- 시간부족(29%) 유저를 위한 마이크로 학습

**근거 데이터**:
- 해지사유 "모든 레슨 완료" 5.2%
- 해지사유 "시간 부족" 29% — 미니레슨으로 대응 가능
- AI 기능 활용도 개선과 시너지
