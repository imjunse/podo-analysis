## PM/PO 당장 실행 우선순위 TOP 3

> 작성일: 2026-02-25
> 기준: ROI + 긴급도 + 실행 용이성

### 1순위: 미납(FAILED) 439건 회수 시스템 (예상 매출 회복: 월 2,000만원+)

**왜 1순위인가**: 이미 결제 의사가 있었던 유저의 돈을 못 받는 것. 신규 획득보다 전환율이 압도적으로 높다.

**즉시 실행 (이번 주)**
- [ ] 결제 실패 즉시 인앱 알림 + 푸시 발송
- [ ] D+1: "결제 수단을 확인해주세요" 카카오/SMS
- [ ] D+3: "수강권이 곧 정지됩니다" 긴급 알림
- [ ] D+7: 마지막 안내 + 간편 결제 수단 변경 링크

**다음 주**
- [ ] 미납 현황 실시간 모니터링 대시보드
- [ ] 결제 수단 만료 사전 감지 (만료 30일 전 알림)

**성공 지표**: 미납 439건 → 300건 이하 (30% 감소 목표)

### 2순위: 체험 레슨 노쇼 33% → 20% 이하 (예상 전환 증가: 월 +100건)

**왜 2순위인가**: 체험 등록까지 왔는데 33%가 안 오는 것. CPA 41,441원 들여 획득한 유저가 증발.

**즉시 실행**
- [ ] D-1 체험 레슨 리마인드 (카카오 + 푸시 + SMS)
- [ ] 체험 레슨 시간 변경을 1탭으로 쉽게 (현재 변경 프로세스 점검)
- [ ] 노쇼 시 자동 재예약 제안 ("다음 시간에 다시 해볼까요?")

**다음 주**
- [ ] 노쇼 원인 설문 (체험 예약 후 미참여자 대상)
- [ ] 체험 전 기대감 콘텐츠 발송 ("첫 레슨에서 이런 걸 배워요")

**성공 지표**: 체험 완료율 66.7% → 80%+

### 3순위: Organic 트래픽 복원 착수 (중기, 하지만 지금 시작해야 함)

**왜 3순위인가**: Paid 90% 의존은 구조적 리스크. Organic이 1년간 71% 붕괴. 지금 시작 안 하면 회복에 6개월+ 소요.

**이번 주 (진단)**
- [ ] Google Search Console 연동 + 현재 SEO 상태 진단
- [ ] 과거 Organic 트래픽 상위 페이지 목록 추출 (어떤 콘텐츠가 유입을 만들었는지)
- [ ] 유튜브 채널 현황 점검 (2,338 → 334로 86% 감소 원인)

**다음 주 (계획)**
- [ ] 월 4건 블로그 콘텐츠 발행 계획 수립 (일본어 학습, 영어 회화 팁 등)
- [ ] 유튜브 숏폼 주 1회 발행 테스트
- [ ] 인스타그램 교육 콘텐츠 리그램 전략

**성공 지표**: 3개월 후 Organic 비중 6% → 12%+
