

> 코드베이스 기반 정책 정리 (2026-03-25 기준)

## 예약 정책

| 항목 | 규칙 |
|------|------|
| 예습(Pre-study) 완료 조건 | 최소 **8분** 이상 수행 |
| 수업 입장 가능 시점 | 수업 시작 **10분 전**부터 |
| 수업 입장 마감 | 수업 종료 시각 + **2분**까지 |
| 예약 변경 마감 | 수업 시작 **2시간 전**까지 |

## 취소 정책 (핵심: 2시간 기준)

수업 시작까지 남은 시간에 따라 3단계로 나뉜다.

| 시간 구간 | 상태 | 환불 | 페널티 |
|-----------|------|------|--------|
| **2시간 이상** 전 | `CANCEL` + 새 수업 생성 | X (수업권 복구 + 재예약) | 없음 |
| **1~2시간** 전 | `CANCEL` | 환불 없음 | 수업권 차감 (일반) / 72시간 예약 금지 (무제한) |
| **1시간 이내** | `CANCEL_PAID` | 환불 있음 | 수업권 차감 (일반) / 72시간 예약 금지 (무제한) |

## 무제한 수업권 vs 일반 수업권 페널티 차이

- **일반 수업권** (`originCount != 999`): 수업권 횟수가 차감됨
- **무제한 수업권** (`originCount == 999`, `penaltyYn == "Y"`): **72시간 예약 금지** (hold) 적용

## 노쇼(No-show) 처리

| 상태 | 설명 |
|------|------|
| `NOSHOW_S` | 학생 노쇼 |
| `CANCEL_NOSHOW_T` | 튜터 노쇼 |
| `NOSHOW_BOTH` | 양쪽 모두 노쇼 |

- 불참 버튼 표시: 수업 시작 **12시간 전**부터 수업 종료까지
- 불참 처리 가능: 수업 시작 **10분 후**부터
