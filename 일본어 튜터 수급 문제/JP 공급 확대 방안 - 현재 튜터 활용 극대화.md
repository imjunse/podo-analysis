## 1. 배경

- 신규 바이링구얼(일한) JP 튜터 고용이 잘 안 되는 상황
- 현재 튜터 대부분 부업 → 저녁 시간대에만 수업 가능
- 일본 hiring season으로 JP 튜터 이탈 증가 중
- **질문: 현재 튜터를 활용해서 수업 슬롯을 어떻게 더 늘릴 수 있는가?**

## 3. 해결 방안

### 3.1 "이왕 접속했으면 1슬롯 더" 인센티브 ⭐ 최우선

**현행**: 튜터가 1-2슬롯만 열고 퇴장 (3월 중 1,779건, 전체의 35%)
**변경**: 3슬롯 이상 열면 보너스 지급

| 시뮬레이션 | 수치 |
|-----------|------|
| 1-2슬롯 튜터-일 (3월) | 1,779건 |
| 이 중 30%만 1슬롯 추가 시 | **+534슬롯/월** |
| 일평균 추가 | **+약 18슬롯/일** |

#### 구체적 인센티브 설계

- **3슬롯 보너스**: 하루 3슬롯 이상 열면 슬롯당 수당 10-20% 할증
- **연속 슬롯 보너스**: 30분 쉬고 연달아 3개 하면 추가 보상
- **부업 튜터 심리**: "어차피 컴퓨터 켰으니 하나 더 하자" 유도

#### 비용 추정

- 슬롯당 수당이 X원이라면, 10% 할증 = 추가 슬롯당 0.1X 비용
- 월 534슬롯 × 0.1X = 약 53.4X원 추가 비용
- 이 비용으로 풀부킹 완화 → 학생 이탈 방지 → ROI 긍정적

### 3.2 비인기 시간대 프리미엄 수당

**현행**: 모든 시간대 동일 수당
**변경**: 오전/오후 시간대 수당 20-30% 프리미엄

| 시간대 | 현재 일평균 | 튜터 수 | 수요 상태 |
|--------|-----------|---------|----------|
| 밤 (20-24시) | 422 | 399 | 풀부킹 |
| 오후 (12-18시) | 215 | 293 | 여유 |
| 오전 (09-12시) | 119 | 238 | **여유 큼** |

- **낮 전용 튜터 112명**: 월 평균 23.4회밖에 안 함 (활용률 낮음)
- 수당 프리미엄으로 이 튜터들이 주 2-3회만 더 열면 → **+300슬롯/월**
- 동시에 학생 수요를 낮으로 유도하면 밤 풀부킹 완화

#### 일평균 추가 슬롯: +약 12/일

### 3.3 저활동 튜터 리텐션 프로그램

**타겟**: 주 1-5회 튜터 160명
**문제**: 이미 등록된 튜터인데 거의 활동하지 않음
**목표**: 주 2회만 더 하게 하면 +320슬롯/주 = +46/일

#### 인센티브 설계

- **최소 활동 보장 보너스**: 주 5회 이상이면 기본급 + 보너스
- **스트릭 보상**: 4주 연속 주 5회 이상 → 추가 인센티브
- **160명 중 절반만 반응해도** → **일 +23슬롯**

#### 비용 추정

- 80명 × 주 2회 추가 × 보너스 = 비교적 저비용
- Hiring season 동안 기존 튜터 활성화가 신규 채용보다 효율적

### 3.4 수업 시간 효율화 (피크 시간대 20분 옵션)

**현행**: 25분 수업 + 5분 간격 = 30분 사이클
**변경**: 피크 시간대(20-24시)에 20분 수업 옵션 추가

| 비교 | 25분 (현행) | 20분 (옵션) |
|------|-----------|-----------|
| 1시간당 슬롯 | 2개 | 3개 |
| 피크 4시간 (20-24시) | 8슬롯/튜터 | 12슬롯/튜터 |
| 일 슬롯 (400튜터 기준) | ~1,050 | ~1,400 |
| 일평균 추가 | - | **+350** |

> 효과가 가장 크지만 **서비스 품질과의 트레이드오프** 큼.
> 전면 적용이 아닌 **피크 시간대 20분 옵션**으로 선택권 제공 방식 검토.

## 4. 종합 시뮬레이션

### 가격 인상 없이, 현재 튜터만으로 가능한 슬롯 추가

| 방안 | 일평균 추가 | 구현 난이도 | 비용 |
|------|-----------|-----------|------|
| 1. +1슬롯 인센티브 | **+18** | 하 | 수당 할증 |
| 2. 비인기 시간 프리미엄 | **+12** | 하 | 수당 할증 |
| 3. 저활동 튜터 리텐션 | **+23** | 중 | 보너스 |
| 4. 20분 수업 옵션 | **+350** | 상 | 서비스 변경 |
| **1+2+3 합산** | **+53/일** | - | - |

### 스마트 매칭과의 시너지

```
공급 확대 (+53슬롯/일)
    +
스마트 매칭 (예약 변경 30% 감소 → 슬롯 체감 가용성 증가)
    =
────────────────────────────────────────
현재 일 슬롯:         ~1,053
공급 추가:             ~1,106  (+5%)
+ 스마트 매칭 효과:    실질 가용 +10-15% 체감
────────────────────────────────────────
풀부킹 알림:          ~770회/일 → 예상 ~400-500회/일
```

## 7. 풀부킹 대응 종합 전략 요약

3개 분석 문서를 관통하는 전체 전략:

```
수요 측 (학생)                    공급 측 (튜터)
─────────────                   ─────────────
1. 스마트 매칭                    1. +1슬롯 인센티브
   → 변경 30% 감소                  → +18슬롯/일

2. 변경 쿨다운                    2. 저활동 리텐션
   → 슬롯 회전율 개선                → +23슬롯/일

3. 저빈도 보호 슬롯               3. 비인기 시간 프리미엄
   → 이탈 방지                      → +12슬롯/일

(최후 수단)                      (장기 검토)
4. 단계별 요금제                  4. 20분 수업 옵션
```

> 가격 인상과 튜터 선택(비장의 무기)은 모두 보존하면서, 현재 자원으로 최대한 풀부킹을 완화하는 전략.
