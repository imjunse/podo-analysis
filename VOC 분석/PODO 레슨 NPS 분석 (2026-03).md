## Part C. 종합 인사이트

### 핵심 발견: "교정 품질"이 만족과 불만의 핵심 축

```
만족: 태도(1위) > 재미(2위) > 교정(3위이지만 합산 시 최대)
불만: 교정 부족(1위, 33%) > 성의없음(2위, 29%) > 소음(3위, 19%)
```

긍정/부정 모두 **교정 관련 피드백이 핵심 변수**.
- 교정을 잘 하면 → veryGood
- 교정이 부족하면 → bad + 차단 요청

### 구조적 문제

1. **부정률 7~8% 고착** — 6개월간 개선 없이 매월 ~150건 부정 수업
2. **교정 부족이 만성 1위** — 10월 이후 6개월 연속 1위
3. **소음 문제 미해결** — 월 20건대 꾸준 (튜터 근무환경)

### 신규 vs 기존 시스템 비교

| 항목 | 기존 (4단계) | 신규 (5단계 ★) |
|------|:---:|:---:|
| 부정률 | **7.5%** | 2.4% |
| 사유 선택 | 단일 | 복수 |
| 자유텍스트 | 없음 | 있음 |
| 차단 기능 | 없음 | 있음 |

> [!tip] 신규 시스템 주의점
> - 5단계 척도 + 복수 선택 → 부정률이 낮아 보이지만 실제 불만은 비슷
> - 자유텍스트 + 차단 기능은 개선 → 더 풍부한 정성 데이터 수집 가능
> - 축적 시 기존 시스템 대비 더 유의미한 분석 가능
