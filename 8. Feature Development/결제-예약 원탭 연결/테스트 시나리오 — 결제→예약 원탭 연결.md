# 테스트 시나리오: 결제→예약 원탭 연결

**PRD**: [[PRD — 결제 완료→예약 원탭 연결]]
**총 시나리오**: 12개
**커버리지**: Happy Path 3 / 엣지 케이스 4 / 에러 1 / GA4 트래킹 2 / 회귀 2

---

## Happy Path

### S1: 신규 유저 — 모바일 앱에서 결제 후 첫 수업 예약
**우선순위**: Critical
**전제 조건**: 수업 이력 없는 신규 유저, 수강권 미보유 상태

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 수강권 결제 완료 | 결제 성공 화면 노출 |
| 2 | 결제 성공 화면 확인 | "구매가 완료되었어요!" + "첫 수업을 예약해 보세요" 문구 표시 |
| 3 | Primary CTA 확인 | **"첫 수업 예약하기"** 버튼 노출 (기존 "예습하러 가기" 아님) |
| 4 | "첫 수업 예약하기" 탭 | `/lessons/regular?referrer=payment-success` 로 이동 |
| 5 | 레슨 선택 화면 확인 | **"첫 수업을 골라보세요"** 안내 배너 상단 노출 |
| 6 | 레슨 카드 선택 | 예약 화면 진입. URL에 `referrer=payment-success` 포함 |
| 7 | 시간/튜터 선택 후 예약 완료 | 예약 완료 화면 노출 |

**완료 조건**: 예약 완료, 마이페이지에서 예약 내역 확인 가능

---

### S2: 신규 유저 — 모바일 웹에서 결제 후 앱으로 이동
**우선순위**: Critical
**전제 조건**: 모바일 웹에서 결제 완료, 앱 설치되어 있음

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 모바일 웹에서 결제 완료 | 결제 성공 화면 노출 |
| 2 | 부제목 확인 | "첫 수업을 예약해 보세요" 표시 (기존 "앱으로 이동해 볼까요?" 아님) |
| 3 | Primary CTA 확인 | **"앱으로 이동하기"** 버튼 (기존과 동일 — 앱 유도 유지) |
| 4 | "앱으로 이동하기" 탭 | 딥링크로 앱 열림 |
| 5 | Secondary CTA 확인 | **"나중에 할게요"** → `/subscribes` (기존 "홈으로 이동하기" 아님) |

**완료 조건**: 앱 진입

---

### S3: 신규 유저 — 데스크톱에서 결제 (변경 없음 확인)
**우선순위**: High
**전제 조건**: 데스크톱 브라우저에서 결제 완료

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 데스크톱에서 결제 완료 | 결제 성공 화면 노출 |
| 2 | 화면 구성 확인 | QR코드 + "홈으로 이동하기" 버튼 (기존과 동일) |
| 3 | QR코드 스캔 | 앱 다운로드/열기 정상 동작 |
| 4 | "홈으로 이동하기" 클릭 | 홈으로 이동 |

**완료 조건**: 기존 데스크톱 동작과 동일

---

## 엣지 케이스

### S4: 체험권 결제 유저 — 자동 배정된 수업 안내
**우선순위**: Critical
**전제 조건**: 체험권 결제 (API 응답에 `extras.classId` 존재)

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 체험권 결제 완료 | 결제 성공 화면 노출 |
| 2 | Primary CTA 확인 | **"수업 확인하기"** 버튼 ("첫 수업 예약하기" 아님) |
| 3 | "수업 확인하기" 탭 | `/reservation` (예약 확인 화면)으로 이동 |
| 4 | 예약 확인 화면 | 자동 배정된 체험 수업 정보 표시 |

---

### S5: 재구매 유저 — 수업 이력 있는 경우
**우선순위**: High
**전제 조건**: 기존에 수업을 1회 이상 수강한 유저가 추가 수강권 구매

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 수강권 추가 결제 완료 | 결제 성공 화면 노출 |
| 2 | Primary CTA 확인 | **"수업 목록 보기"** 버튼 ("첫 수업 예약하기" 아님) |
| 3 | "수업 목록 보기" 탭 | `/subscribes` (레슨 목록)로 이동 |

---

### S6: Secondary CTA — "나중에 할게요" 선택
**우선순위**: Medium
**전제 조건**: 신규 유저, 모바일 앱에서 결제 완료

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 결제 성공 화면에서 "나중에 할게요" 탭 | `/subscribes` (레슨 목록)로 이동 |
| 2 | 레슨 목록 화면 확인 | 안내 배너 **미노출** (referrer 파라미터 없으므로) |

---

### S7: referrer 파라미터 전파 확인
**우선순위**: High
**전제 조건**: 신규 유저, "첫 수업 예약하기" CTA로 진입

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 레슨 선택 화면 URL 확인 | `referrer=payment-success` 포함 |
| 2 | 레슨 카드 선택 → 예약 화면 URL 확인 | `referrer=payment-success` 포함 |
| 3 | 예약 완료 후 뒤로가기 | `/home`으로 이동 (기존 예약 목록이 아님) |

---

## 에러 시나리오

### S8: 모바일 웹 — 앱 미설치 시 딥링크 폴백
**우선순위**: High
**전제 조건**: 모바일 웹에서 결제 완료, 앱 **미설치**

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | "앱으로 이동하기" 탭 | 딥링크 시도 |
| 2 | 앱이 없으므로 실패 | 폴백: 앱스토어/플레이스토어로 이동 |

## GA4 이벤트 트래킹

### S9: CTA 클릭 이벤트 발화 확인
**우선순위**: High
**전제 조건**: GA4 디버그 모드 활성화

| Step | 액션 | 기대 이벤트 |
|:----:|------|------------|
| 1 | "첫 수업 예약하기" 탭 | `payment_success_cta_click` { cta_type: "book_first_class", platform: "app" } |
| 2 | "나중에 할게요" 탭 | `payment_success_cta_click` { cta_type: "later", platform: "app" } |
| 3 | 데스크톱에서 "홈으로 이동하기" 클릭 | `payment_success_cta_click` { cta_type: "home", platform: "desktop" } |

---

### S10: curriculum_type 파라미터 정확성
**우선순위**: Medium

| Step | 액션 | 기대 이벤트 |
|:----:|------|------------|
| 1 | Regular 수강권 결제 후 CTA 클릭 | `curriculum_type: "regular"` |
| 2 | Premium 수강권 결제 후 CTA 클릭 | `curriculum_type: "premium"` |

## 회귀 테스트

### S11: 기존 결제 완료 이벤트 영향 없음
**우선순위**: Critical

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 결제 완료 | 기존 GA4 `purchase` 이벤트 정상 발화 |
| 2 | 결제 성공 화면 | 축하 애니메이션 정상 동작 |
| 3 | 결제 정보 표시 | 결제 금액, 수강권 정보 정상 표시 |

---

### S12: 기존 문구 제거/변경 확인
**우선순위**: Medium

| Step | 액션 | 기대 결과 |
|:----:|------|----------|
| 1 | 모바일 앱 결제 성공 화면 | "예습하러 가기" 문구 **미노출** (→ "첫 수업 예약하기"로 대체됨) |
| 2 | 모바일 앱 결제 성공 화면 | "홈으로 이동하기" 문구 **미노출** (→ "나중에 할게요"로 대체됨) |
| 3 | 모바일 웹 결제 성공 화면 | "앱으로 이동하기" 문구 **유지** (변경 없음) |
| 4 | 모바일 앱/웹 부제목 확인 | 기존 플랫폼별 문구 **미노출** (→ "첫 수업을 예약해 보세요"로 통합됨) |
