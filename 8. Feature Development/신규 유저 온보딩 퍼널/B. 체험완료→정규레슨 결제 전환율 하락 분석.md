## Executive Summary

### **분석 목적**
"체험레슨 완료 to 정규레슨 결제 볼륨 하락 원인의 명확한 진단"

### **핵심 문제**
- **신규결제 일평균**: 33.3건/일(1월) → 28.7건/일(3월) **-14% 하락**
- **근본 원인**: **체험완료→결제 전환율 하락** (44.24% → 31.25%, -13%p)

### **주요 발견**
1. **세일즈 액션 변경**: 체험레슨 완료 후 미결제자 유저 지속 탭핑중이었으나 중단한 액션이 가장 메인 원인
2. **프로모션 변경 직접 타격**: 1/8 변경 후 즉시 44.24% → 40.05% (-4.2%p)
3. **지속적 하락 트렌드**: 이후 31.25%까지 총 -13%p 하락
4. **1월 시즈널 프리미엄 포함**: 44.24%는 신년 효과(+7~8%p) + 프로모션 효과 혼재
5. **2월 페이백 프로모션 한계**: 44.05% 일시 회복하지만 7일 만에 재하락
6. **마케팅비 감소는 부차 요인**: 2월 마케팅비 감소에도 체험완료는 증가하여 메인 요인은 아님


### **해결 방향**
**세일즈팀 D+4~7 전화 액션 정상화**가 최우선 과제

## **프로모션 변경 임팩트**

### **1월 프로모션 변경사항**
- **1/1~7**: 회원가입 쿠폰 변경 + **신년 시즈널 효과** → **44.24% 달성**
- **1/8 이후**: 쿠폰 혜택 축소 → **40.05%** (즉시 -4.2%p 하락)
- **1월 중순**: **36.6% 안정화** (시즈널 효과 소멸 후 순수 프로모션 효과)

### **시사점**
1. **프로모션 민감도 높음**: 변경 즉시 전환율 타격
2. **시즈널 프리미엄 존재**: 44.24%에는 신년 효과(추정 +7~8%p) 포함
3. **현실적 기준선**: 시즈널 제외 시 36~37% 수준
4. **현재 프로모션 한계**: 31.25% 수준에 머물러
